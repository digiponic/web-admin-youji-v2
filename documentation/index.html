Kebutuhan Spesifikasi

Dalam menjalankan sistem aplikasi ini yang anda butuhkan yaitu :

  1. Komputer atau laptop dengan sistem operasi Windows / Linux / Mac
  2. Koneksi internet yang stabil
  3. Web Browser untuk menjalankan sistem aplikasi (cth: Google Chrome or Mozilla Firefox)

Mulai #kembali ke atas

Hal pertama yang anda lakukan untuk mengakses sistem aplikasi yaitu membuka web browser di komputer anda, kemudian ketikkan alamat url http://app.digiponic.co.id/youji dan tekan enter. Maka akan muncul tampilan awal seperti pada gambar berikut.

Halaman Login

Halaman login diisi dengan nama pengguna dan kata sandi yang terdaftar dalam sistem.

  • Langkah 1 - Ketikkan nama pengguna.
  • Langkah 2 - Ketikkan kata sandi sesuai dengan pengguna.
  • Langkah 3 - Tekan enter atau klik tombol masuk.

Halaman Awal

Halaman awal menampikan informasi data yang dapat digunakan sebagai analisa singkat untuk rencana atau tindakan lain yang dapat dilakukan.

Halaman berikut ini akan menampilkan data tipe yang digunakan sebagai master awal database.

Ubah Tipe

Hapus Tipe

Halaman berikut ini merupakan tampilan peringatan sebelum anda menghapus data tipe, klik Tidak jika ingin membatalkan atau klik Ya jika ingin menghapus data tipe tersebut.

Detail Tipe

Halaman berikut ini akan menampilkan informasi detail tipe.

Submenu Tipe

Transaksi#Kembali Ke Atas

Hal pertama yang anda lakukan untuk mengakses menu transaksi yaitu dengan klik menu master transaksi yang berada dibawah menu transaksi pada daftar menu sebelah kiri pada tampilan web browser di komputer anda. Setelah di klik maka akan muncul sub menu.
Terdapat beberapa sub menu yang dapat digunakan yaitu:

  1. Penjualan
  2. Pembelian

1. Penjualan

Pada menu master Penjualan digunakan untuk membuat data penjualan dimana transaksi penjualan ini memiliki beberapa fungsi yakni menambah transaksi, kemudian mengubah data transaksi, dapat print out transaksi berupa invoice penjualan, kemudian melihat dan mengubah status transaksi, dan dapat menghapus data transaksi penjualan

List Daftar Transaksi Penjualan

Halaman berikut ini akan menampilkan daftar detail transaksi.

Tambah Transaksi Penjualan

Halaman berikut ini akan menampilakan transaksi penjualan dimana anda dapat menambahkan pelanggan, kemudian detail produk anda dapat menambahkan diskon/potongan harga di setiap produk atau total keseluruhan produk

List Produk Transaksi Penjualan

Berikut ini adalah list produk yang dapat di tambahkan pada saat melakukan transaksi penjualan

Ubah Transaksi Penjualan

Halaman berikut ini merupakan tampilan form ubah data transaksi penjualan

Detail Transaksi Penjualan

Halaman berikut ini merupakan tampilan peringatan sebelum anda menghapus data tipe, klik Tidak jika ingin membatalkan atau klik Ya jika ingin menghapus data tipe tersebut.

Proses Transaksi Penjualan

Halaman berikut ini merupakan tampilan list transaksi penjualan dimana terdapat fungsi status sebelah kanan simbol mobil berwarna merah yang memberikan info bahwa barang siap di proses. fungsi proses pengiriman tersebut dapat dirubah dengan cara klik buttonketika barang terkirim kepada customer
Selain terdapat fungsi untuk proses terdapat fungsi button untuk mencetak struk yang invoice penjualan

Laporan Penjualan

Halaman berikut ini merupakan tampilan laporan transaksi penjualan dimana laporan tersebut dapat di ekspor dalam bentuk Microsoft Excel, PDF, CSV.

Hapus Transaksi Penjualan

Halaman berikut ini merupakan tampilan peringatan sebelum anda menghapus data transaksi penjualan, klik Tidak jika ingin membatalkan atau klik Ya jika ingin menghapus data transaksi penjualan tersebut.
